# Hospital Database Project

## Системийн тайлбар
Энэ төсөл нь эмнэлгийн мэдээллийн системийн өгөгдлийн сан юм.  
Өвчтөн, эмч, үзлэг болон жорын мэдээллийг хадгална.

## Хүснэгтүүд

- doctors – эмч
- appointments – үзлэг
- prescriptions – жор
